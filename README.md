# Vanilla UI
A meteor addon to change the appearance of 6b6t UI to appear more vanilla-like. 

## Installation
- Head over to [Releases](https://github.com/azecx/vanilla-ui/releases)
- Install the latest one.
- Put it in your `.minecraft/mods` folder along with [Meteor](https://meteorclient.me/)

## Features
Customizable colors for each unique rank and greentext in chat.
